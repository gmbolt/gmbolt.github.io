# Description

This is a GitHub-hosted website which I am using to facilitate access of custom themes in Marp.

Usually in VS code one would need to specify a relative path to the CSS files where themes are stored. This is all well and good unless you want to locate your Marp documents within various different folders, for example, the presentation of a particular project might want to be stored in that project's folder. Finding the theme CSS files with respect to a given folder can be tedious and thus a more ideal solution would be where there is some absolute location for CSS files. 

This can be achieved by storing the CSS files on a public website, since these can then be accessed via a URL. Thus I created this repository.
