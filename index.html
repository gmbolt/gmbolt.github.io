My Marp Themes

This is a basic website hosted via GitHub Pages to facilitate access of CSS files for custom themes in the Marp presentation framework.

The current options are

  1. my-theme - URL: https://gmbolt.github.io/my-theme.css - a very basic edit of the Uncover theme;
  2. my-theme-color - URL: https://gmbolt.github.io/my-theme-color.css - similar to my-theme but with color bar at bottom.

If using VS Code one can use custom themes. To do so the proces goes as follows

  1. Go to the Settings tab and search "Marp" to find all Marp settings;
  2. Look for the input titled "Markdown> Marp: Themes";
  3. Add the URL of your desired theme.

Now, for example, with https://gmbolt.github.io/my-theme.css added, I could write "theme: my-theme" in Marp and theme defined by my-theme.css will be applied.
