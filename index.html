My Marp Themes

This is a basic website hosted via GitHub Pages to facilitate access of CSS files for custom themes in the Marp presentation framework.

The current options are

  1. my-theme - URL: gmbolt.github.io/my-theme.css - a very basic edit of the Uncover theme;
  2. my-theme-color - URL: gmbolt.github.io/my-theme-color.css - similar to my-theme but with color bar at bottom.
